When an ANDON screen is turned on, it will look for configured shifts that start in the next 30 minutes and select that shift. If no shift is scheduled to start in the next 30 minutes, it will use the current shift if there is one. Production made before the official start is early production, and production after the shift end is considered overtime. If no shifts are found now or in the next 30 minutes, it will assume a new shift that lasts 8 hours just started and record production out of schedule.
